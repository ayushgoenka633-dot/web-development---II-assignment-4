# BookNest: Smart Book Finder

A React + Vite capstone project using the Open Library API.

## Features
- Book search using public API
- Save/remove favourite books (CRUD with localStorage)
- React Router pages
- Context API state management
- Search, sort and pagination
- Dashboard with charts
- Dark mode
- Error and loading states

## Run locally
```bash
npm install
npm run dev
```

## Netlify settings
- Build command: `npm run build`
- Publish directory: `dist`
