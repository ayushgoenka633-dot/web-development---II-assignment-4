# BookNest: Smart Book Finder

BookNest is a React-based Smart Book Discovery Dashboard using the Open Library API.

## Features

- React + Vite frontend
- React Router navigation
- Context API for state management
- Open Library API integration using Axios
- Search with debounced API calls
- Sort by relevance, newest books, and edition count
- Pagination with Previous/Next buttons
- CRUD operations using saved library in localStorage
- Dark mode toggle
- Dashboard analytics
- Error handling
- Responsive Tailwind CSS UI

## Tech Stack

- React
- Vite
- JavaScript ES6+
- React Router DOM
- Context API
- Axios
- Tailwind CSS
- Open Library API

## Run Locally

```bash
npm install
npm run dev
```

## Build for Deployment

```bash
npm run build
```

## Deployment

Deploy on Netlify or Vercel. Build command: `npm run build`. Publish directory: `dist`.
